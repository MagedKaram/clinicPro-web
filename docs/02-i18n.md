# i18n (AR/EN)

## الهدف

- النظام من الأول bilingual: عربي وإنجليزي.

## Setup الحالي

- `next-intl` + `middleware` لتحديد الـ locale.
- رسائل: `web/messages/ar.json` و `web/messages/en.json`.
- Routes: `web/src/app/[locale]/**`.

## قواعد

- ممنوع نصوص hard-coded داخل JSX.
- كل صفحة لها namespace واضح (مثلاً `reception.*`, `doctor.*`).
